# PG Atlas SDK (@scf/pg-atlas-sdk)

[![NPM Version](https://img.shields.io/npm/v/@scf/pg-atlas-sdk)](https://www.npmjs.com/package/@scf/pg-atlas-sdk)
[![License](https://img.shields.io/badge/license-Apache--2.0-blue)](./LICENSE)

The official **PG Atlas SDK** for interacting with the PG Atlas API. Built for developers building on the Stellar and Soroban ecosystem within the Public Goods Atlas framework.

## 🚀 Installation

```bash
npm install @scf/pg-atlas-sdk
# or
pnpm add @scf/pg-atlas-sdk
# or
yarn add @scf/pg-atlas-sdk
```
